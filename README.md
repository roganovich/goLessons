# Курс Golang

## Пишем REST API сервис на Go
1. Config
    - cleanenv
2. Logger
    - slog
3. Storage
    - sqllite
4. Router
    - chi
5. Server

**Консольные команды**

* Запустить приложение
```bash  
go run main.go
```

* Создать модуль в приложение
```bash 
go mod init go-lessons/junior-rest-api
```

* Собрать приложение
```bash 
go build
```

* Запустить приложение как бинарник
```bash 
./junior-rest-api
```
